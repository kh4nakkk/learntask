JavaScript Objects

  • Objects are collections of key-value pairs.
  • Example: 
    let car = { color: "red", make: "Toyota", year: 2022 };
  • Access object properties using dot notation (car.color) or bracket notation (car["make"]).
  • Modify properties or add new ones like this: 
    car.model = "Corolla";
  • Objects can contain functions, called methods: 
    
let person = {
    firstName: "John",
    lastName: "Doe",
    fullName: function() {
        return this.firstName + " " + this.lastName;
    }
};

JavaScript Arrays

  • Arrays are ordered lists of values.
  • Example: 
    let colors = ["red", "green", "blue"];
  • Access array elements using their index: 
    console.log(colors[0]); // Output: "red"
  • Add new elements using push(): 
    colors.push("yellow"); // Adds "yellow" to the end
  • Arrays can store any data type, even other arrays or objects!
  • Example of a mixed array: 
    let mixedArray = ["hello", 42, { key: "value" }, [1, 2, 3]];
