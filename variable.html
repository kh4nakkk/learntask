Variables

Introduction

  • Variables and Data types are the foundation concepts of a a Js program.
  • Can also be called the building block for storing and manipulating data.

Types of Variables

var

  • Declares a variable
  • General space " " cant be used hence an underscore ( _ ) can be used instead.

  • For eg.
    [IN CASE OF A NUMBER]

    var x=10;

    Here the variable x has stored the value 10 and when called via console.log(x), the value returned will be 10.

    [IN CASE OF A STRING]

    var name="Khanak";

    Here too, the value stored is the string declared in the vaiable x.
