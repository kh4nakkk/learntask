Operators

Introduction

  • Basically any Arithematic or Logical computing is done via using operators.

Types of Operators

Arithematic Operators

  • (+): Used for addition.
  • (-): used for substration
  • (*): used for multiplication
  • (**): exponential
  • (/): used for division
  • (%): used to get the remainder or "modulus"
  • (++): used to increas any number value by +1
  • (--): used to decrease any number value by -1

Assignment Operators

  • Assignment (=): Assign value
  • Addition (+=): Add and assign
  • Subtraction (-=): Subtract and assign
  • Multiplication (*=): Multiply and assign
  • Division (/=): Divide and assign
  • Modulus (%=): Remainder and assign

    • Comparison Operators

    • Equality (==): Compare values
    • Strict Equality (===): Compare strictly
    • Inequality (!=): Not equal
    • Strict Inequality (!==): Not strictly equal

    Logical Operators

    • Logical AND (&&): Both true
    • Logical OR (||): Either true
    • Logical NOT (!): Negate value

    Ternary Operator

    • (?): conditional operator
