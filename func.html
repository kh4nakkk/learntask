Functions

Introduction

  • A JavaScript function is a block of code designed to perform a particular task.
  • It would only be executed when it is called.

Function Declaration

A function is declared using the function keyword.we use these to declare a function :

  • A name
  • then using () after naming the function
  • A pair of curly braces {} containing the code that runs when the function is called.

Function call

  • to call the function we call its name. (will see in a example)

function with Parameters

  • A function can take parameters, which are like inputs that you can pass into the function. This gives us he freedom to input any value when calling the function

Function with return

  • Functions can also return a value using the return keyword. This is useful when you want the function to calculate something and give the result back.

Pros of using functions

  • Reuseability: We can call the function as many times depending on our will.
  • Simplify: it simplifies complex codes
  • Clean code: It makes your code look neat.
